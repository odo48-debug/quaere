# Quaere Engine (Local-First Agent Database)

**A Private, In-Browser PostgreSQL Engine & Explorer for AI Agents.**

Quaere is a specialized data storage and management layer designed to give AI agents (like [OpenClaw](https://github.com/OpenClaw/OpenClaw)) a persistent, relational memory directly in the browser. 

Unlike traditional agent memory (flat Markdown files or remote cloud databases), Quaere provides a full SQL-capable engine that runs entirely on the user's device, ensuring absolute privacy and zero-latency queries.

## 🚀 Key Features

- **💾 Local-Only PostgreSQL**: Powered by [PGlite](https://pglite.dev/), providing a full PostgreSQL 17 engine running in a Web Worker via WASM. Data stays in your browser's IndexedDB.
- **🏗️ Agent-Centric Explorer**: A clean, modern interface to audit and manage the structured data being generated by your agents in real-time.
- **⚡ Live Queries**: Reactive UI that updates instantly when an agent (or an external process) writes to the local database.
- **🔒 Privacy-by-Default**: No cloud, no tracking, no data exfiltration. Every byte of sensitive information lives and dies on the local machine.
- **🛠️ Open Architecture**: Designed to be cloned, improved, and integrated into local agentic workflows.
- **🔗 Agent Bridge**: Integrated `BroadcastChannel` for seamless communication with external AI agents (like OpenClaw).

## 🔗 Agent Bridge (Interoperability)
Quaere includes a `BroadcastChannel` bridge that allows external AI agents or browser extensions to interact with the database without direct integration.

**Channel Name:** `quaere_agent_bridge`

**Example Request (from an Agent):**
```javascript
const channel = new BroadcastChannel('quaere_agent_bridge');
channel.postMessage({
  type: 'SQL_QUERY',
  id: 'unique_id_123',
  sql: 'INSERT INTO memory (task, status) VALUES ($1, $2)',
  params: ['Research PGlite', 'completed']
});

channel.onmessage = (event) => {
  if (event.data.requestId === 'unique_id_123') {
    console.log('Result:', event.data.data);
  }
};
```

## 🛠️ Technology Stack

- **Frontend:** React, TypeScript, Tailwind CSS
- **Database Engine:** [PGlite](https://pglite.dev/) (PostgreSQL 17.4 via WASM)
- **Persistence:** IndexedDB / Origin Private File System (OPFS)
- **Worker Isolation:** Dedicated Web Workers for non-blocking DB operations

## 📂 Project Structure

- `src/App.tsx`: The heart of the database explorer.
- `src/lib/pglite.ts`: Logic for table management and SQL generation.
- `src/components/DatabaseView.tsx`: The primary data grid for auditing records.
- `src/components/DatabaseSchemaView.tsx`: A visual representation of your relational schema.

   ```bash
   cd quaere-engine
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run locally:**
   ```bash
   npm run dev
   ```
